Add CSS Grid controls to Elementor

== Description ==
The **Xmedia Elementor Grid** plugin enhances Elementor\'s CSS Grid capabilities, allowing you to precisely control grid placement without custom CSS.

### Key Features:
* Assign elements to specific grid columns and rows directly in Elementor.
* Fully responsive controls for desktop, tablet, and mobile.
* Lightweight and seamlessly integrated with Elementor.

### How It Works:
1. Install and activate the plugin.
2. Open the Elementor editor and select a section or element.
3. Find \"CSS Grid controls by Xmedia\" under the layout options.
4. Use the **Grid Column** and **Grid Row** fields to define placement:
   * Example: `1 / 3` places the element from column 1 to 3.
   * Example: `span 2` makes the element stretch over 2 columns.

== Installation ==
Note : This plugin works with Elementor. Make sure you have Elementor installed.

1. Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress

== Frequently Asked Questions ==

= What does the Xmedia Elementor Grid plugin do? =
The plugin adds CSS Grid controls to Elementor, allowing you to assign elements to specific grid columns and rows without writing custom CSS.

= Do I need coding knowledge to use this plugin? =
No! The plugin integrates seamlessly into Elementor and provides simple input fields for defining grid placement.

= Can I use this plugin with other Elementor add-ons? =
Yes, this plugin is designed to work alongside other Elementor add-ons without conflicts.

= Does the plugin support responsive layouts? =
Absolutely! The grid controls are fully responsive, allowing you to define settings for desktop, tablet, and mobile.

= What happens if I deactivate the plugin? =
If you deactivate the plugin, the custom grid settings will no longer be applied, but your content will remain unaffected.

= Are there any performance concerns? =
No, the plugin is lightweight and doesn’t add unnecessary overhead to your website.
